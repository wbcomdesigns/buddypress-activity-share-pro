# Sharing Activities to Social Networks

Every activity in your community's feed has a **Share** button. Clicking it opens a small menu where you can send the activity to any social network your site administrator has enabled.

## What you will see

After clicking the Share button below an activity, a dropdown opens with two sections:

- **Reshare** — share the activity inside your community (covered in the next article)
- **Social network buttons** — icons for each external platform your admin turned on

Each social network button opens a new browser tab or window that takes you to that platform's sharing page, with the activity link and title already filled in.

## Supported networks

The following networks can appear in the Share menu. Your admin chooses which ones to show and in what order.

| Network | What it does |
|---|---|
| Facebook | Opens the Facebook Share dialog with the activity link |
| X (formerly Twitter) | Opens a pre-filled post on X with the activity title and link |
| LinkedIn | Opens the LinkedIn share form with the activity link |
| WhatsApp | Opens WhatsApp with the activity link ready to send |
| Telegram | Opens Telegram with the activity link ready to forward |
| Pinterest | Opens the Pinterest "Create Pin" page with the activity link |
| Reddit | Opens the Reddit "Submit a Link" form with the activity title and link |
| WordPress | Opens the WordPress.com Press This bookmarklet with the activity link |
| Pocket | Saves the activity link to your Pocket reading list |
| Bluesky | Opens a pre-filled Bluesky compose window with the activity link |
| E-mail | Opens your email client with a subject and body pre-filled |
| Copy Link | Copies the activity link to your clipboard — no new tab opens |

Not all of these will be visible on every site. The site administrator controls which networks are enabled.

## How to share step by step

1. Find the activity you want to share in the activity feed.
2. Click the **Share** button (the share icon with the word "Share") in the activity's action bar.
3. The Share menu opens. Look for the social network icons below the Reshare button.
4. Click the icon for the network you want to use.
5. A new tab opens on that platform. Finish posting there, then return to your community.

For **Copy Link**, no new tab opens. You will see a small "Link Copied!" confirmation tooltip. Paste the link wherever you need it.

For **E-mail**, your default email client opens with the subject and body already written. Add your recipient addresses and send.

## Sharing when you are not logged in

If your site administrator has enabled sharing for visitors, you can share activities to social networks without logging in. The Reshare option (share inside the community) always requires an account.

## Share count

A number appears next to the Share button showing how many times that activity has been shared — both to external networks and internally within the community. The count updates automatically.
