# Requirements

Before installing BuddyPress Activity Share Pro, confirm that your site meets these requirements.

## WordPress and PHP

| Requirement | Minimum version |
|---|---|
| WordPress | 5.0 |
| PHP | 7.4 |
| BuddyPress | 8.0 |

PHP 8.0, 8.1, and 8.2 are all supported. The plugin has been tested up to WordPress 6.9.

## BuddyPress or BuddyBoss Platform

The plugin requires either:

- **BuddyPress** — the free plugin available from wordpress.org. Version 8.0 or later.
- **BuddyBoss Platform** — the BuddyBoss commercial platform. The plugin includes full compatibility for BuddyBoss-specific features.

The BuddyPress Activity component must be active. This plugin adds sharing controls to the activity stream, so if you have the Activity component disabled in BuddyPress settings, nothing will appear.

## Free Version Conflict

If you have the free **BuddyPress Activity Social Share** plugin installed, it will be deactivated automatically when you activate this Pro version. The two plugins cannot run at the same time.

## Theme Compatibility

The plugin works with any WordPress theme. Extra styling is included for:

- **BuddyX** (free) and **BuddyX Pro**
- **Reign Theme**

If you use a different theme, the share buttons will appear using the plugin's default styles.

## What the Plugin Does Not Require

- No API keys for any social network — sharing links open on the social platform itself, so no external API credentials are needed.
- No CDN or external image service.
- No WooCommerce or any e-commerce plugin.

## Next Step

Once your site meets these requirements, follow the [Installation](./installation.md) guide to get the plugin running.
